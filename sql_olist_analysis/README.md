# Olist 巴西电商数据分析项目（SQL）
基于巴西真实电商公开数据集，使用 MySQL 完成完整的业务指标分析，适合数据分析师简历项目。

## 项目背景
分析平台订单、用户、商品、物流数据，输出业务指标与运营建议。

## 分析内容
- 整体规模：订单量、GMV、用户数
- 月度订单与GMV趋势
- 城市订单TOP10
- 商品品类销量与销售额
- 用户复购率
- 客单价分析
- 订单状态分布
- 物流时效分析

## 技术栈
- MySQL
- SQL（聚合、JOIN、CTE、子查询）

## 数据集
Kaggle Olist Brazilian E-Commerce Dataset
