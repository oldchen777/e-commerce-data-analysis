# 电商销售数据分析项目

## 项目概述
使用Python和pandas对电商销售数据进行清洗、分析与可视化，挖掘销售趋势与热销商品，为业务决策提供数据支持。

## 技术栈
Python
pandas
Matplotlib

## 核心步骤
1.数据读取并清洗:处理缺失值、异常值
2.指标计算:总销售额、客单价、客户总数
3.多维度分析:按月、国家、商品等统计销量
4.可视化:销售趋势、热销商品前十等可视化图表

## 如何运行
1.克隆仓库：git clone https://github.com/oldchen777/e-commerce-data-analysis.git
2.  安装依赖：pip install pandas matplotlib openpyxl
3.  运行代码：python 电商销售数据分析.py

## 项目结论
1.英国是主要市场，贡献了大部分销售额。
2.部分月份存在明显销售高峰，可针对性制定促销策略。
3.少数商品销量远高于其他，可重点备货与推广。
